# LogoBriefGuide Astro Starter

This is a drop-in starter pack for **LogoBriefGuide.com**.

## Use it like this
1. Create a fresh Astro site:
   ```bash
   npm create astro@latest .
   ```
2. Choose a minimal starter.
3. Install the sitemap integration:
   ```bash
   npx astro add sitemap
   ```
4. Replace the generated files with the files in this starter pack.
5. Run locally:
   ```bash
   npm install
   npm run dev
   ```
6. Deploy to Cloudflare Pages.
7. Add both:
   - `logobriefguide.com`
   - `www.logobriefguide.com`
8. Make `https://logobriefguide.com` the canonical host.
9. Submit the sitemap to Google Search Console and Bing Webmaster Tools.

## Portfolio defaults used here
- Canonical host: non-www
- Trailing slash: always
- Sitemap enabled
- Dynamic robots.txt route
- Simple shared layout
- Legal pages included
- Clarity supported but off by default
